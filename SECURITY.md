# Security Policy

## Status

tsumugi is pre-1.0 / experimental. It has not been audited and is not
recommended for use with sensitive data in production.

## Reporting a vulnerability

If you discover a security vulnerability, please open a private security
advisory on GitHub:

https://github.com/archfill/tsumugi/security/advisories/new

Do not file public issues for security concerns.

I aim to acknowledge reports within 7 days. There is no formal disclosure
SLA at this stage of the project.
